Au premier démarrage du logiciel, il vous sera demandé de vous enregistrer (ce n'est pas obligatoire, sachez toutefois que l'accès à certaines fonctionnalités du logiciel requiert un compte administrateur).

Si vous choisissez de vous enregistrer, vous devrez saisir un identifiant (le logiciel vérifiera si celui-ci est déjà utilisé), un mot de passe et spécifier si l'utilisateur créé est Administrateur.

Une fois ces données saisies et vérifiées, l'utilisateur sera automatiquement sauvegardé dans un fichier nommé Utilisateurs.xml. Ce fichier regroupe les identifiants, mots de passe et options réglées pour chaque utilisateur créé.

Une fois ceci fait, vous accédez à la page principale du logiciel (en mode "écran scindé") où vous pourrez commencer vos figures géométriques.



Attention! Le logiciel stocke les utilisateurs créés dans un fichier nommé Utilisateurs.xml. Fichier à ne surtout pas supprimer!
