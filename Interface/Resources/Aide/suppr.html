Le logiciel vous permet de supprimer un utilisateur, pour cela il vous suffit de vous connecter en tant qu'administrateur et saisir l'identifiant de l'utilisateur à supprimer.

Après avoir vérifié l'existence de l'identifiant, le logiciel se chargera de supprimer les données de l'utilisateur stocké dans le fichier Utilisateurs.xml (voir ici).



Attention! Cette opération est irréversible, réfléchissez-bien avant de supprimer un utilisateur!
