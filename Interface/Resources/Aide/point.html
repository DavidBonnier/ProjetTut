Lorsque vous vous trouvez en mode Plein Écran géométrie, vous avez accès à l'outil de création de point.

Pour utiliser cet outil, il vous faut cliquer sur l'icone correspondante (en forme de croix) dans la barre d'outils à gauche de votre écran. Un premier clic sur cet icone active la création, un second clic n'importe où dans l'espace de travail créera le point à l'emplacement du curseur.



Remarque: Un second clic sur le bouton de création des points vous permet de quitter ce mode.
