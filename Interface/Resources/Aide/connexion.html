Au démarrage du logiciel, il ne vous ait jamais demandé de vous connecter. Cette fonctionnalité est accessible dans le menu Fichier ou en utilisant le raccourci Ctrl+B.

Si vous choississez de vous connecter, vous devrez saisir un identifiant et un mot de passe. Il va de soi que la validité de l'identifiant spécifié ainsi que la concordance avec le mot de passe saisi seront vérifiés par le logiciel. Si vous êtes administrateur vous aurez alors accès à toutes les fonctionnalités du logiciel (ajout/suppression d'un utilisateur) .

Si vous souhaitez modifier les paramètres spécifiques à un élève, vous n'avez pas besoin de vous connecter en tant qu'administrateur, ces réglages sont accessibles par mot de passe (le logiciel vérifie si ce mot de passe est bien celui d'un administrateur).

Le fichier regroupant tous les utilisateurs (voir ici) va alors chargé les paramètres qu'un administrateur aura réglé pour l'utilisateur dont il est question.

Une fois ceci fait, vous accédez à la page principale du logiciel (en mode "écran scindé") où vous pourrez commencer vos figures géométriques.
