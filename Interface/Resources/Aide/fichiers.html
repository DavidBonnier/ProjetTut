Après avoir créé vos figures et les avoir intégrées dans vos cahiers vous pouvez bien entendu sauvegarder votre travail. Il vous est aussi possible d'ouvrir un fichier sauvegardé au préalable.

Pour sauvegarder votre travail, il vous suffit de selectionner l'une des deux options "Sauvegarder" ou "Sauvegarder sous" dans le menu déroulant "Fichier". Si vous n'avez jamais sauvegardé votre travail, vous serez automatiquement redirigé vers la fonction "Sauvegarder sous" qui vous permettra de sauvegarder votre fichier où bon vous semble avec le nom de votre choix.

Pour ouvrir un fichier, vous devrez cliquer sur "Ouvrir" dans le menu déroulant "Fichier" et selectionner un fichier valide. Celui-ci sera alors chargé et vous retrouverez vos créations et annotations inchangées.



A noter que les fonctionnalités présentées ci-avant sont accessibles par le biais de raccourcis claviers (F12 pour Sauvegarder Sous, Ctrl+S pour Sauvegarder, Ctrl+O pour Ouvrir un document).
